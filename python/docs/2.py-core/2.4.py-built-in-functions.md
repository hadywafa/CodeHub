## **ğŸ“Œ 1. Type Conversion Functions**

These functions **convert data types**.

### **ğŸ”¹ `int()` - Convert to Integer**

```python
print(int("42"))   # 42 (String to Integer)
print(int(3.9))    # 3  (Float to Integer, truncates decimal part)
print(int(True))   # 1  (Boolean to Integer)
print(int(False))  # 0  (Boolean to Integer)
```

---

### **ğŸ”¹ `float()` - Convert to Float**

```python
print(float("3.14"))  # 3.14 (String to Float)
print(float(10))      # 10.0 (Integer to Float)
```

---

### **ğŸ”¹ `str()` - Convert to String**

```python
print(str(100))      # '100' (Integer to String)
print(str(3.14))     # '3.14' (Float to String)
```

---

### **ğŸ”¹ `bool()` - Convert to Boolean**

```python
print(bool(1))       # True
print(bool(0))       # False
print(bool(""))      # False (Empty string is False)
print(bool("Hello")) # True (Non-empty string is True)
```

---

### **ğŸ”¹ `list()` - Convert to List**

```python
print(list("hello"))     # ['h', 'e', 'l', 'l', 'o'] (String to List)
print(list((1, 2, 3)))   # [1, 2, 3] (Tuple to List)
```

---

### **ğŸ”¹ `tuple()` - Convert to Tuple**

```python
print(tuple([1, 2, 3]))   # (1, 2, 3) (List to Tuple)
print(tuple("hello"))     # ('h', 'e', 'l', 'l', 'o')
```

---

### **ğŸ”¹ `set()` - Convert to Set**

```python
print(set([1, 2, 3, 3, 4]))   # {1, 2, 3, 4} (Remove duplicates)
print(set("banana"))         # {'a', 'b', 'n'} (Unique characters)
```

---

### **ğŸ”¹ `dict()` - Convert to Dictionary**

```python
print(dict([("name", "Alice"), ("age", 25)]))
# {'name': 'Alice', 'age': 25} (List of Tuples to Dictionary)
```

---

## **ğŸ“Œ 2. Input & Output Functions**

### **ğŸ”¹ `print()` - Display Output**

```python
print("Hello, World!")   # Hello, World!
```

#### **Printing Multiple Values**

```python
print("Name:", "Alice", "Age:", 25)  # Name: Alice Age: 25
```

#### **Using Custom Separator**

```python
print("apple", "banana", "cherry", sep=" - ")
# Output: apple - banana - cherry
```

#### **Printing with End Parameter**

```python
print("Hello", end=" ")
print("World!")
# Output: Hello World! (Prevents new line)
```

---

### **ğŸ”¹ `input()` - Get User Input**

```python
name = input("Enter your name: ")
print("Hello", name)
```

#### **Getting Numerical Input**

```python
age = int(input("Enter your age: "))  # Convert input to int
print("Next year, you will be", age + 1)
```

## **ğŸ“Œ 3. Mathematical Functions**


---

### **ğŸ”¹ `pow()` - Power Calculation**

### **ğŸ”¹ `sum()` - Sum of Elements**

## **ğŸ“Œ 4. Sequence Functions**

### **ğŸ”¹ `len()` - Get Length**

```python
print(len("Hello"))   # 5
print(len([1, 2, 3])) # 3
```

---

### **ğŸ”¹ `sorted()` - Sort Elements**

```python
print(sorted([3, 1, 2]))  # [1, 2, 3]
print(sorted("python"))  # ['h', 'n', 'o', 'p', 't', 'y']
```

---

### **ğŸ”¹ `reversed()` - Reverse Elements**

```python
print(list(reversed([1, 2, 3])))  # [3, 2, 1]
```

---

### **ğŸ”¹ `enumerate()` - Get Index & Value**

```python
names = ["Alice", "Bob"]
for index, name in enumerate(names):
    print(index, name)
# Output:
# 0 Alice
# 1 Bob
```

---

### **ğŸ”¹ `zip()` - Combine Two Sequences**

```python
names = ["Alice", "Bob"]
ages = [25, 30]

print(list(zip(names, ages)))
# [('Alice', 25), ('Bob', 30)]
```

---

## **ğŸ“Œ 5. Logical & Utility Functions**

### **ğŸ”¹ `all()` - Check if All Elements are True**

```python
print(all([True, 1, "hello"]))  # True
print(all([True, 0, "hello"]))  # False
```

---

### **ğŸ”¹ `any()` - Check if Any Element is True**

```python
print(any([False, 0, "hello"]))  # True
print(any([False, 0, ""]))  # False
```

---

### **ğŸ”¹ `id()` - Get Memory Address of Object**

```python

print(id(x))
```

---

### **ğŸ”¹ `type()` - Get Type of Object**

```python
print(type(10))      # <class 'int'>
print(type("Hello")) # <class 'str'>
```

---

### **ğŸ”¹ `isinstance()` - Check Data Type**

```python
print(isinstance(10, int))  # True
print(isinstance("hello", str))  # True
print(isinstance(3.14, (int, float)))  # True
```
