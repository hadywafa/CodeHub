## **ğŸŸ¢ ORDERED & MUTABLE: LIST (`list`)**

### **ğŸ”¹ What is a List?**

A **list** is an **ordered, mutable** sequence of elements.


### **ğŸ”¹ Common List Methods**

```python

```

### **ğŸ”¹ Slicing Lists**

```python

```

### **ğŸ”¹ Iterating Through a List**

âœ… **Best for dynamic collections** where elements need frequent changes.

### **ğŸ”¹ Why Use Tuples?**

âœ… **Faster than lists** (since they are immutable)

### **ğŸ”¹ Common Tuple Methods**

âœ… Tuples **cannot be modified**, but you can **convert them into lists**:

## **ğŸŸ  UNORDERED & MUTABLE: SET (`set`)**

### **ğŸ”¹ What is a Set?**

A **set** is an **unordered, mutable** collection of **unique elements**.

```python

```

### **ğŸ”¹ Common Set Methods**

```python

print(unique_numbers.difference({1, 2}))  # Elements not in second set
```

âœ… Sets are **faster than lists** for membership tests (`x in my_set`).
