## **ğŸ“Œ 3. Variable Reassignment & Memory Behavior**

Reassigning a variable **does not change** the original object; instead, it creates a **new object reference**.

### **ğŸ”¹ Example: Integer Reassignment**

```python
x = 100
print(id(x))  # Memory address of 100

x = 200  # Reassignment creates a new object
print(id(x))  # New memory address for 200
```

âœ… **`x` now points to a new object (`200`), and `100` may be garbage collected.**

---

### **ğŸ”¹ Example: List Mutation (Mutable Type)**

```python
nums = [1, 2, 3]
print(id(nums))  # Memory address before mutation

nums.append(4)  # Modifies the same object
print(id(nums))  # Same memory address
```

âœ… **Lists are mutable**; changes **modify the existing object in memory**.

## **ğŸ“Œ 4. Memory Optimization: Variable Caching (Interning)**

Python **optimizes memory usage** using **object interning** for:  
âœ… **Small integers (`-5` to `256`)**  
âœ… **Short strings**

### **ğŸ”¹ Example: Integer Interning**

```python

b = 42

```

âœ… **Both point to the same memory location.**

---

### **ğŸ”¹ Example: String Interning**

```python

print(id(s1) == id(s2))  # True (cached)

```

âœ… **Short strings are interned**, but long strings **may not be**.

## **ğŸ“Œ 6. Shallow vs. Deep Copying in Memory**

When copying objects, Python creates either:

- **Shallow Copy** (New reference, same nested objects)
- **Deep Copy** (Full duplication)

### **ğŸ”¹ Example: Shallow Copy (`copy.copy()`)**

```python
import copy

original = [1, [2, 3], 4]
shallow_copy = copy.copy(original)

shallow_copy[1][0] = 99  # Modifies original list too!
print(original)  # [1, [99, 3], 4]
```

âœ… **Shallow copy creates a new list, but inner lists are shared.**

---

### **ğŸ”¹ Example: Deep Copy (`copy.deepcopy()`)**

```python
deep_copy = copy.deepcopy(original)
deep_copy[1][0] = 42  # Does NOT affect the original list

print(original)  # [1, [99, 3], 4]
```

âœ… **Deep copy creates an entirely separate object.**

## **ğŸ“Œ 7. Variable Scope & Lifetime in Memory**

Variables exist in different **scopes**:

- **Local** (Inside a function)
- **Global** (Outside a function)
- **Nonlocal** (Enclosing function in a nested function)

### **ğŸ”¹ Example: Local vs Global Scope**

```python

def modify():
    x = 5  # Local variable
    print("Inside:", x)

modify()
print("Outside:", x)  # Global 'x' is unaffected
```

âœ… **Local and global variables are stored in different memory spaces.**
