## **ðŸ“Œ 1. Integer (`int`)**

### **ðŸ”¹ What is an Integer?**

An **integer** (`int`) is a **whole number** (positive, negative, or zero) without a decimal point.

```python
x = 10      # Positive integer
y = -5      # Negative integer
z = 0       # Zero
```

### **ðŸ”¹ Integer Operations**

Python supports **arithmetic operations** on integers:

```python
a = 10
b = 3

print(a + b)  # Addition -> 13
print(a - b)  # Subtraction -> 7
print(a * b)  # Multiplication -> 30
print(a / b)  # Division -> 3.3333 (Returns float)
print(a // b) # Floor Division -> 3 (No decimal)
print(a % b)  # Modulus -> 1 (Remainder)
print(a ** b) # Exponentiation -> 1000 (10Â³)
```

### **ðŸ”¹ Common Methods for Integers**

```python
num = 123

print(abs(num))      # Absolute value -> 123
print(pow(2, 3))     # Exponentiation -> 8 (2Â³)
print(bin(num))      # Binary representation -> '0b1111011'
print(hex(num))      # Hexadecimal representation -> '0x7b'
print(oct(num))      # Octal representation -> '0o173'
```

### **ðŸ”¹ Notes & Best Practices**

âœ… **Use integers for counting, indexing, and mathematical operations**.  
âœ… **Avoid using leading zeros** (`010` is invalid in Python 3).  
âœ… **Python integers are arbitrary precision**, meaning they can store extremely large values.

## **ðŸ“Œ 2. Floating-Point Numbers (`float`)**

### **ðŸ”¹ What is a Float?**

A **float** represents a **decimal number** (fractional or real numbers).

```python

temperature = -5.7
big_number = 1.2e3  # 1.2 Ã— 10Â³ -> 1200.0
```

### **ðŸ”¹ Float Operations**

```python

print(a + b)  # Addition -> 7.7
print(a - b)  # Subtraction -> 3.3
print(a * b)  # Multiplication -> 12.1
print(a / b)  # Division -> 2.5
```

### **ðŸ”¹ Common Float Methods**

```python

print(round(num, 2))  # Round to 2 decimal places -> 3.14
print(float(10))      # Convert int to float -> 10.0
```

### **ðŸ”¹ Notes & Best Practices**

âœ… **Floating-point precision issues exist** (`0.1 + 0.2 != 0.3` due to binary storage).  
âœ… Use the **`decimal` module** for high-precision calculations.

```python

```

## **ðŸ“Œ 3. Boolean (`bool`)**

### **ðŸ”¹ What is a Boolean?**

A **boolean** represents **True or False** values.

```python
is_active = True
is_deleted = False
```

### **ðŸ”¹ Boolean Operations**

```python
print(5 > 3)   # True
print(5 == 3)  # False
print(bool(0)) # False
print(bool(1)) # True
print(bool("")) # False (Empty string)
print(bool("Hello")) # True (Non-empty string)
```

### **ðŸ”¹ Common Boolean Methods**

Booleans **inherit methods from integers**, since `True` is equivalent to `1` and `False` to `0`.

```python
print(int(True))  # 1
print(int(False)) # 0
print(True + True)  # 2 (1+1)
```

### **ðŸ”¹ Notes & Best Practices**

âœ… Use **`bool()` to check truthiness of a value**.  
âœ… **All non-zero numbers, non-empty strings, and non-empty structures evaluate to `True`**.

## **ðŸ“Œ 4. String (`str`)**

### **ðŸ”¹ What is a String?**

A **string** is a sequence of **characters** enclosed in `""` or `''`.

```python
name = "Alice"
message = 'Hello, World!'
multiline = """This is a
multiline string."""
```

### **ðŸ”¹ String Operations**

```python
first = "Hello"
second = "World"

print(first + " " + second)  # Concatenation -> "Hello World"
print(first * 3)  # Repetition -> "HelloHelloHello"
```

### **ðŸ”¹ Common String Methods**

```python
text = "python programming"

print(text.upper())   # "PYTHON PROGRAMMING"
print(text.lower())   # "python programming"
print(text.title())   # "Python Programming"
print(text.strip())   # Removes whitespace
print(text.replace("python", "JavaScript")) # Replace word
print(len(text))  # Length of string -> 18
```

### **ðŸ”¹ Notes & Best Practices**

âœ… Use **f-strings (`f""`) for formatting** (best practice in Python 3.6+).

```python
name = "Alice"
age = 25
print(f"My name is {name} and I am {age} years old.")
```

âœ… Strings are **immutable** (cannot be changed after creation).

## **ðŸ“Œ 5. Dictionary (`dict`)**

### **ðŸ”¹ What is a Dictionary?**

A **dictionary** stores **key-value pairs**.

```python
person = {
    "name": "Alice",
    "age": 25,
    "city": "New York"
}
```

### **ðŸ”¹ Common Dictionary Methods**

```python
print(person.keys())   # Get all keys
print(person.values()) # Get all values
print(person.get("age")) # 25
print(person.items())  # Get all key-value pairs
person["email"] = "alice@example.com"  # Add a new key-value pair
```

### **ðŸ”¹ Notes & Best Practices**

âœ… Keys must be **unique** and **immutable** (strings, numbers, tuples).  
âœ… Use `get()` to avoid **KeyError**.

```python
print(person.get("phone", "Not Found"))  # Returns default value
```

## **ðŸ“Œ 6. Set (`set`)**

### **ðŸ”¹ What is a Set?**

A **set** is an **unordered collection** of **unique elements**.

```python

print(nums)  # {1, 2, 3, 4} (Duplicates removed)
```

### **ðŸ”¹ Common Set Methods**

```python
nums.add(5)   # Add element
nums.remove(2) # Remove element
print(nums)
```

### **ðŸ”¹ Notes & Best Practices**

âœ… Use **sets for uniqueness** (e.g., removing duplicates from a list).
