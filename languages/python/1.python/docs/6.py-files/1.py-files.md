## Python File Handling Guide

This document provides an overview of reading, writing, and appending to files in Python, based on best practices for resource management.

### 1. Reading from a File

Reading text from a file follows three main steps:

1. **Open** the file using `open()`.
2. **Read** the content.
3. **Close** the connection.

**Example:**

```python
file = open('shopping_list.txt', 'r')
contents = file.read()
file.close()
print(contents)
```

*Note: Use the `.split()` method to turn a single string into a list of items (04:16).*

### 2. Writing to a File

Use mode `'w'` to create a new file or overwrite an existing one.

**Example:**

```python
items = "milk\neggs\nbread"
file = open('groceries.txt', 'w')
file.write(items)
file.close()
```

*Warning: Opening in `'w'` mode immediately erases existing file contents (07:33).*

### 3. Appending to a File

Use mode `'a'` to add content to the end of an existing file without deleting current data.

**Example:**

```python
with open('groceries.txt', 'a') as file:
    file.write("\nyogurt")
```

### 4. Resource Management (Context Managers)

Using the `with` keyword ensures that files are automatically closed, preventing memory buffer issues and resource waste.

**Example:**

```python
with open('groceries.txt', 'a') as file:
    file.write("\nbananas")
```

### 5. Working with File Paths

- **Relative Paths:** By default, files are created in the current project folder.
- **Absolute Paths:** Define an exact location on your computer.

**Example (Windows):**

```python
with open(r'C:\Users\Desktop\groceries.txt', 'w') as file:
    file.write("milk")
```

*Tip: Use an `r` before the string on Windows to handle backslashes correctly (12:36).*
