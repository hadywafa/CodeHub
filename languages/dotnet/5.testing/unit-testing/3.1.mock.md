## ğŸ­ Moq Key Features

### 1. Setup with Conditions

```csharp

    .Returns(new User { Name = "Guest" });
```

- `It.IsAny<int>()` = wildcard matcher.

---

### 2. Verify Method Calls

```csharp

```

Checks method was called once with parameter `5`.

---

### 3. Throwing Exceptions

```csharp

```

---

### 4. Properties

```csharp

```
