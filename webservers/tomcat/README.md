# Tomcat Web Server

> Tomcat is a web server and servlet container that implements the Java Servlet and JavaServer Pages technologies.

## Installation

### 1. Download the latest version of Tomcat from the [official website](https://tomcat.apache.org/download-21.cgi).

### 2. Extract the zip file to a desired location.

### 3. Set the `JAVA_HOME` environment variable to the location of the JDK.

### 4. Start the Tomcat server by running the `startup.sh` script.
