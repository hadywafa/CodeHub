## ğŸ—‚ **2. Types of Variables**

There are **three** types of variables in shell scripting:

### ğŸ”¹ **2.1. Local Variables**

- Declared inside a script or shell session.
- Not accessible outside the current shell.

```bash
greeting="Hello"
echo "$greeting"
```

### ğŸ”¹ **2.2. Environment Variables**

- Used by the system and can be accessed globally.
- Set using `export` and accessed in sub-processes.

```bash
export MY_VAR="Hello World"
echo "$MY_VAR"
```

### ğŸ”¹ **2.3. Special Variables (Built-in)**

| Variable        | Description                               |
| --------------- | ----------------------------------------- |
| `$0`            | Name of the script                        |
| `$1, $2, $3...` | Command-line arguments                    |
| `$#`            | Number of arguments                       |
| `$@`            | All arguments (separately quoted)         |
| `$*`            | All arguments (as a single string)        |
| `$$`            | Process ID of the script                  |
| `$?`            | Exit status of the last command           |
| `$!`            | Process ID of the last background command |

## ğŸ–¥ **4. Reading User Input (`read` Command)**

Shell scripts can interactively ask for user input.

### âœ… **4.1. Basic `read` Command**

```bash

```

### âœ… **4.2. Reading Multiple Inputs**

```bash

```

### âœ… **4.3. Silent Input (`-s` for Passwords)**

```bash

```

### âœ… **4.4. Input with Timeout (`-t`)**

```bash

```

### âœ… **4.5. Using Default Values**

```bash

```

### ğŸ”¹ **5.1. Command Substitution**

### ğŸ”¹ **5.2. Arithmetic Operations**

### ğŸ”¹ **5.3. String Manipulation**

| Operation | Syntax                 | Example             |
| --------- | ---------------------- | ------------------- |
| Length    | `${#var}`              | `${#name}`          |
| Substring | `${var:offset:length}` | `${name:2:3}`       |
| Replace   | `${var/old/new}`       | `${name/Alice/Bob}` |

```bash

echo "Substring: ${name:6:9}"
echo "Replaced: ${name/Alice/Bob}"
```

### ğŸ”¹ **5.4. Checking if a Variable is Set**
