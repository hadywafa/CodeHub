## **ðŸ“Œ 8. Inheritance in PowerShell**

Inheritance allows **a child class to reuse code from a parent class**.

### **ðŸ”¹ Example: Inheriting a Class**

```powershell
class Animal {
    [string]$Name

    Animal([string]$name) {
        $this.Name = $name
    }

    [string] Speak() {
        return "Some generic sound"
    }
}

class Dog : Animal {
    Dog([string]$name) : base($name) {}

    [string] Speak() {
        return "$($this.Name) says: Woof!"
    }
}

$myDog = [Dog]::new("Buddy")
Write-Output $myDog.Speak()
```

âœ… The `Dog` class **inherits** from `Animal` and **overrides** `Speak()`.

## **ðŸ“Œ 9. Polymorphism (Method Overriding)**

Polymorphism allows **methods to behave differently in derived classes**.

### **ðŸ”¹ Example: Overriding a Method**

```powershell

    [string]$Type

    Vehicle([string]$type) {
        $this.Type = $type
    }

    [string] Move() {
        return "$($this.Type) is moving..."
    }
}

class Car : Vehicle {
    Car([string]$type) : base($type) {}

    [string] Move() {
        return "$($this.Type) is driving on the road!"
    }
}

$bike = [Vehicle]::new("Bicycle")
$tesla = [Car]::new("Tesla Model 3")

Write-Output $bike.Move()   # Bicycle is moving...
Write-Output $tesla.Move()  # Tesla Model 3 is driving on the road!
```

âœ… The `Move()` method **behaves differently** in `Vehicle` and `Car`.
