## **üìå 1. Conditional Statements in PowerShell**

Conditional statements allow scripts to make **decisions** based on conditions.

### **üîπ if-else Statement**

The `if-else` statement evaluates a condition and executes the appropriate block.

#### **Syntax:**

```powershell
if (<condition>) {
    # Code to execute if condition is true
} elseif (<another_condition>) {
    # Code if the second condition is true
} else {
    # Code if none of the conditions are met
}
```

#### **Example: Check a Number**

```powershell
$number = 10

if ($number -gt 10) {
    Write-Output "Number is greater than 10"
} elseif ($number -eq 10) {
    Write-Output "Number is exactly 10"
} else {
    Write-Output "Number is less than 10"
}
```

‚úÖ **Comparison Operators in PowerShell:**

| Operator | Meaning                  |
| -------- | ------------------------ |
| `-eq`    | Equal to                 |
| `-ne`    | Not equal to             |
| `-gt`    | Greater than             |
| `-lt`    | Less than                |
| `-ge`    | Greater than or equal to |
| `-le`    | Less than or equal to    |

---

### **üîπ Switch Statement**

`switch` evaluates an expression and executes the matching case.

#### **Syntax:**

```powershell
switch ($value) {
    "Option1" { Write-Output "Selected Option 1" }
    "Option2" { Write-Output "Selected Option 2" }
    default { Write-Output "No match found" }
}
```

#### **Example: Check User Role**

```powershell
$userRole = "Admin"

switch ($userRole) {
    "Admin" { Write-Output "You have full access." }
    "Editor" { Write-Output "You can edit content." }
    "Viewer" { Write-Output "You can only view content." }
    default { Write-Output "Invalid role." }
}
```

‚úÖ The `default` case runs **if no match is found**.

## **üìå 2. Looping Constructs in PowerShell**

Loops **repeat** a set of commands **until a condition is met**.

---

### **üîπ For Loop (Fixed Iterations)**

The `for` loop runs **a fixed number of times**.

#### **Syntax:**

```powershell
for (<initialization>; <condition>; <increment>) {
    # Code to execute
}
```

#### **Example: Print Numbers from 1 to 5**

```powershell
for ($i = 1; $i -le 5; $i++) {
    Write-Output "Iteration: $i"
}
```

‚úÖ The loop:  
1Ô∏è‚É£ Starts with `$i = 1`  
2Ô∏è‚É£ Checks the condition `$i -le 5`  
3Ô∏è‚É£ Increments `$i++` after each iteration

---

### **üîπ Foreach Loop (Iterate Through Collections)**

The `foreach` loop **iterates over arrays and collections**.

#### **Syntax:**

```powershell
foreach ($item in <collection>) {
    # Code to execute
}
```

#### **Example: Iterate Over an Array**

```powershell
$fruits = @("Apple", "Banana", "Cherry")

foreach ($fruit in $fruits) {
    Write-Output "I like $fruit"
}
```

‚úÖ Used for processing **lists, arrays, and hashtables**.

---

### **üîπ While Loop (Runs Until Condition is False)**

The `while` loop **runs as long as a condition is true**.

#### **Syntax:**

```powershell
while (<condition>) {
    # Code to execute
}
```

#### **Example: Print Numbers Until 5**

```powershell
$i = 1
while ($i -le 5) {
    Write-Output "Number: $i"
    $i++  # Increment the counter
}
```

‚úÖ **Best for cases when the number of iterations is unknown**.

---

### **üîπ Do-While Loop (Runs At Least Once)**

The `do-while` loop **executes the code at least once** before checking the condition.

#### **Syntax:**

```powershell
do {
    # Code to execute
} while (<condition>)
```

#### **Example: Get User Input Until Valid**

```powershell
do {
    $number = Read-Host "Enter a number greater than 10"
} while ($number -le 10)

Write-Output "You entered a valid number: $number"
```

‚úÖ The block **executes once before checking the condition**.

---

### **üîπ Do-Until Loop (Runs Until Condition is True)**

The `do-until` loop **runs until a condition becomes true**.

#### **Syntax:**

```powershell
do {
    # Code to execute
} until (<condition>)
```

#### **Example: Ask for Correct Password**

```powershell
do {
    $password = Read-Host "Enter password"
} until ($password -eq "Secret123")

Write-Output "Access granted!"
```

‚úÖ **Opposite of `do-while`**‚Äîstops when the condition becomes true.

---

## **üìå 3. Breaking and Continuing Loops**

### **üîπ Break (Exit the Loop)**

Stops the loop **immediately**.

#### **Example: Exit Loop When Number is 3**

```powershell
for ($i = 1; $i -le 5; $i++) {
    if ($i -eq 3) { break }
    Write-Output "Iteration: $i"
}
```

‚úÖ The loop **exits at 3**.

---

### **üîπ Continue (Skip Current Iteration)**

Skips **only the current iteration** and continues to the next.

#### **Example: Skip Number 3**

```powershell
for ($i = 1; $i -le 5; $i++) {
    if ($i -eq 3) { continue }
    Write-Output "Iteration: $i"
}
```

‚úÖ The loop **skips 3 but continues with 4 and 5**.

---

## **üìå 4. Try-Catch-Finally (Error Handling in Loops)**

PowerShell provides `try-catch-finally` blocks to **handle exceptions gracefully**.

### **Example: Handling Division by Zero**

```powershell
$numbers = @(5, 10, 0, 20)

foreach ($num in $numbers) {
    try {
        $result = 100 / $num
        Write-Output "Result: $result"
    } catch {
        Write-Output "Error: Cannot divide by zero"
    } finally {
        Write-Output "Processed number: $num"
    }
}
```

‚úÖ The `catch` block **handles errors**, preventing script failure.

---

## **üìå 5. Nested Loops in PowerShell**

Loops can be **nested** to process multi-dimensional data.

### **Example: Nested Loop for Multiplication Table**

```powershell
for ($i = 1; $i -le 3; $i++) {
    for ($j = 1; $j -le 3; $j++) {
        Write-Output "$i x $j = " + ($i * $j)
    }
}
```

‚úÖ **Useful for processing tables, grids, and arrays**.
