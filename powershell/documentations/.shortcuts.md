| **Shortcut**    | **Action**                                                                  |
| --------------- | --------------------------------------------------------------------------- |

## **ðŸ“Œ 10. Customizing PowerShell with Aliases**

You can create **custom shortcuts (aliases)** for frequently used commands.

### **ðŸ”¹ View All Aliases**

```powershell

```

### **ðŸ”¹ Create a Custom Alias**

```powershell

```

âœ… Now typing `ll` runs `Get-ChildItem` (like `ls` in Linux).

### **ðŸ”¹ Remove an Alias**
