## **ğŸ“Œ 8. Using `Begin`, `Process`, and `End` Blocks**

PowerShell functions can include **three execution blocks**:

- **`Begin`** â†’ Runs **once** before processing any input
- **`Process`** â†’ Runs **for each input** provided
- **`End`** â†’ Runs **once** after all processing is done

### **ğŸ”¹ Example: Using Begin, Process, and End**

```powershell

    param ([string[]]$names)

    Begin { Write-Output "Starting Processing..." }

    Process {
        foreach ($name in $names) {
            Write-Output "Processing: $name"
        }
    }

    End { Write-Output "Processing Complete!" }
}

# Call function with multiple values

```

âœ… The function **processes each name separately**.

### **ğŸ”¹ Example: Handling Division by Zero**

```powershell
        Write-Output "Operation completed."
    }
}

```

âœ… **Prevents script failure on division by zero**.
