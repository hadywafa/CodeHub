## **ğŸ“Œ 1. Iterators & Looping Functions**

### **ğŸ”¹ `range()` - Generate a Sequence of Numbers**

âœ… Creates a **range object** (efficient in memory).

```python
print(list(range(5)))       # [0, 1, 2, 3, 4]
print(list(range(2, 10, 2)))  # [2, 4, 6, 8] (start=2, stop=10, step=2)
```

---

### **ğŸ”¹ `map()` - Apply Function to Each Element**

âœ… Applies a function to **each item** in an iterable.

```python
def square(n):
    return n * n

numbers = [1, 2, 3, 4]
print(list(map(square, numbers)))  # [1, 4, 9, 16]
```

âœ… Using **lambda** (anonymous function):

```python
print(list(map(lambda x: x * x, numbers)))  # [1, 4, 9, 16]
```

---

### **ğŸ”¹ `filter()` - Filter Elements Based on Condition**

âœ… Keeps only **items that return True**.

```python
numbers = [10, 25, 30, 40, 55]
even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # [10, 30, 40]
```

---

### **ğŸ”¹ `reduce()` - Apply a Function Repeatedly**

âœ… Used for **cumulative operations** (Requires `functools` module).

```python
from functools import reduce

numbers = [1, 2, 3, 4]
product = reduce(lambda x, y: x * y, numbers)
print(product)  # 24 (1 * 2 * 3 * 4)
```

## **ğŸ“Œ 2. Randomization Functions (`random` module)**

### **ğŸ”¹ `random.random()` - Generate Random Float (0-1)**

```python

```

---

### **ğŸ”¹ `random.randint(a, b)` - Generate Random Integer**

```python

```

---

### **ğŸ”¹ `random.choice()` - Pick Random Element**

```python

```

---

### **ğŸ”¹ `random.sample()` - Pick Multiple Random Items**

```python

print(random.sample(numbers, 2))  # Randomly picks 2 unique items
```

---

### **ğŸ”¹ `random.shuffle()` - Shuffle a List**

```python

print(cards)  # ['Queen', 'King', 'Jack', 'Ace'] (Random order)
```

## **ğŸ“Œ 3. File Handling Functions**

### **ğŸ”¹ `open()` - Open a File**

âœ… Modes:

- `'r'` â†’ Read
- `'w'` â†’ Write (overwrite)
- `'a'` â†’ Append
- `'rb'` / `'wb'` â†’ Binary mode

```python
# Write to a file
file = open("example.txt", "w")
file.write("Hello, World!")
file.close()

# Read a file
file = open("example.txt", "r")
print(file.read())  # Hello, World!
file.close()
```

âœ… **Using `with` (better for handling files automatically):**

```python

    content = file.read()
print(content)
```

## **ğŸ“Œ 6. Memory & Performance Optimization**

### **ğŸ”¹ `enumerate()` - Get Index & Value While Looping**

```python

for index, name in enumerate(names):
    print(index, name)
# Output:
# 0 Alice
# 1 Bob
```

---

### **ğŸ”¹ `zip()` - Combine Two Lists**

```python

ages = [25, 30]

print(list(zip(names, ages)))
# [('Alice', 25), ('Bob', 30)]
```

---

### **ğŸ”¹ `any()` & `all()` - Logical Operations**

```python

print(any(values))  # True (At least one is True)
print(all(values))  # False (Not all are True)
```

---

### **ğŸ”¹ `next()` - Get Next Item from an Iterator**

```python

print(next(numbers))  # 1
print(next(numbers))  # 2
```
