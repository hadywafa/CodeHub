## üß† What Are Operators?

**Operators** are special symbols or keywords that perform operations on variables and values ‚Äî arithmetic, logic, comparisons, bitwise, assignments, etc.

---

## ‚ö° C# Operator Categories

| Category                       | Description                                       |
| ------------------------------ | ------------------------------------------------- |
| üßÆ Arithmetic                  | Math operations: `+`, `-`, `*`, `/`, `%`          |
| üß† Comparison / Relational     | Compare values: `==`, `!=`, `<`, `>`, `<=`, `>=`  |
| üîÅ Logical                     | Boolean logic: `&&`, `\|\|`                       |
| üß© Bitwise                     | Bit-level operations: `&`, `\|`,`^`,`~`,`<<`,`>>` |
| üß∑ Assignment                  | Assign values: `=`, `+=`, `-=`, `*=`              |
| ‚ùî Null & Type Checks          | Handle nulls and types: `??`, `?.`, `is`, `as`    |
| üß¨ Unary / Increment/Decrement | Single operand: `++`, `--`, `+`, `-`, `!`         |

## üßÆ 1. Arithmetic Operators

| Operator | Description         | Example | Result    |
| -------- | ------------------- | ------- | --------- |
| `+`      | Addition            | `5 + 2` | `7`       |
| `-`      | Subtraction         | `5 - 2` | `3`       |
| `*`      | Multiplication      | `5 * 2` | `10`      |
| `/`      | Division            | `5 / 2` | `2` (int) |
| `%`      | Modulus (remainder) | `5 % 2` | `1`       |

### ‚ö†Ô∏è Beware of integer division

```csharp

```

## üß† 3. Comparison / Relational Operators

| Operator | Meaning               | Example  | Result  |
| -------- | --------------------- | -------- | ------- |
| `==`     | Equal                 | `5 == 5` | `true`  |
| `!=`     | Not equal             | `5 != 2` | `true`  |

## üß© 6. Bitwise Operators (Advanced)

> ‚ö†Ô∏è Mostly used in **low-level**, **performance**, or **hardware logic** code.

| Operator                     | Symbol | Description                                        |
| ---------------------------- | ------ | -------------------------------------------------- |
| **AND**                      | `&`    | Performs bitwise AND between two values.           |
| **OR**                       | `\|`   | Performs bitwise OR between two values.            |
| **XOR (Exclusive OR)**       | `^`    | Sets bits to `1` where only one operand is `1`.    |
| **NOT (Bitwise Complement)** | `~`    | Flips all bits (`0` becomes `1`, `1` becomes `0`). |
| **Left Shift**               | `<<`   | Moves bits to the left, multiplying by `2^n`.      |
| **Right Shift**              | `>>`   | Moves bits to the right, dividing by `2^n`.        |

## ‚ùî 7. Null-Coalescing & Null Safety

These are **life-saving operators** in real-world apps to handle `null`.

### `??` ‚Üí If null, return fallback

```csharp
string name = null;
string display = name ?? "Unknown";  // "Unknown"
```

### `?.` ‚Üí Null-safe navigation

```csharp
string name = null;
int? length = name?.Length;  // null, not exception
```

### `??=` ‚Üí Assign only if null

```csharp
name ??= "Hady";
```

## üß¨ 8. Type Checking Operators

| Operator | Usage           | Purpose                     |
| -------- | --------------- | --------------------------- |
| `is`     | `obj is string` | True if object is type      |
| `as`     | `obj as string` | Cast if possible, else null |
| `typeof` | `typeof(int)`   | Get Type metadata           |

### Example

```csharp
object o = "Hello";
if (o is string s) {
    Console.WriteLine(s.ToUpper());
}
```
